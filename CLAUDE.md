## Tech Stack
- Vite + React 18
- TypeScript (altijd, nooit plain JS, nooit `any`)
- Tailwind CSS + shadcn/ui
- Framer Motion (animaties)
- React Router DOM (routing)
- Supabase (backend)
- EmailJS (contactformulier)
- Cal.com embed (afspraken plannen)
- Zod (validatie)
- React Query (data fetching)

## Regels
- Code altijd in het Engels, UI-teksten in het Nederlands
